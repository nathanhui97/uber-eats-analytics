🚀 Quick Start Testing

Follow these steps to test your extension locally before using it on real Uber Eats Manager pages.

1 Load Extension in Chrome

Go to chrome://extensions/ → Enable Developer mode → Load unpacked → Select extension/dist folder

2 Open Mock Test Page

Open extension/test-pages/mock-uber-eats.html in Chrome to test data capture

3 Test Extension

Click the extension icon → Start Capturing Data → Check console for logs
